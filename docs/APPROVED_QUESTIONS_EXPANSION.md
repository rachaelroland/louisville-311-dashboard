## Database Schema

All 53 questions stored in: `l311_approved_questions` table

**Key Fields:**
- `question_text` - The question residents ask
- `answer_text` - Comprehensive, helpful answer
- `category` - Main category (13 total)
- `subcategory` - Specific service area
- `question_type` - how_to, what_is, timeline, tracking
- `keywords` - Array for matching
- `common_variations` - Alternative phrasings
- `typical_urgency` - low, medium, high
- `typical_response_time` - Expected resolution time
- `service_name` - Links to 311 service types
- `times_shown` - Usage tracking
- `times_helpful` / `times_not_helpful` - Feedback counters
